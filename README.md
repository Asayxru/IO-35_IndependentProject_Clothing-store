# Clothing Store (Інтернет-магазин одягу)

Навчальний багатосторінковий frontend-проєкт: адаптивний інтерфейс, каталог одягу з фільтрами/сортуванням, кошик покупок (localStorage) та форма зворотного зв'язку з валідацією.

## Технології
- HTML5
- CSS3 (Flexbox/Grid, media queries)
- JavaScript (DOM events, localStorage)

## Запуск
Відкрийте `index.html` у браузері або використайте локальний сервер (наприклад, Live Server у VS Code).
