# Clothing Store

Навчальний адаптивний веб-застосунок інтернет-магазину одягу.

## Технології

* HTML5
* CSS3 (Flexbox, Grid, медіа-запити)
* JavaScript (DOM API, localStorage)

## Функціонал

* Каталог товарів з фільтрацією та сортуванням
* Кошик покупок з підтримкою localStorage
* Адаптивна навігація та бургер-меню
* Форма зворотного зв’язку з клієнтською валідацією
* Адаптивний дизайн для мобільних пристроїв

## Призначення

Проєкт виконано в навчальних цілях у межах індивідуального завдання.
